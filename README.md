# Transaction Intelligence

A learning-focused transaction analysis project built with Python, NumPy and pandas

## Project objective

The objective of this project is to build a complete data analysis workflow using synthetic financial transaction data

The project will cover:

- Data inspection
- Data cleaning
- Data transformation
- Table joins
- Customer level analysis
- Time-series features
- Statistical anomaly detection
- Performance comparisons between pandas, NumPy and Python loops
